Core Features

📦

Transaction Management

Complete waste transaction lifecycle with CRUD operations and status tracking.

  • Create transactions with line items
  • Status workflow (pending → approved)
  • File attachments & photos
  • Mobile QR code input
🤖

AI-Powered Auditing

Automated transaction validation using Google Vertex AI with two-phase processing.

  • Image analysis & material detection
  • Weight anomaly detection
  • Price validation
  • Duplicate detection
📊

GRI Sustainability Reporting

GRI 306 (Waste) standards compliance with automated data aggregation.

  • GRI 306-1: Waste Impacts
  • GRI 306-2: Management
  • GRI 306-3: Waste by Type
  • PDF report generation
🌍

G360 AI ESG Consulting

Interactive AI assistant for sustainability guidance and recommendations.

  • ESG score analysis
  • Improvement recommendations
  • Compliance guidance
  • Carbon footprint insights
⚙️

Setup & Configuration

Multi-tenant organization management with hierarchical structure.

  • Role-based access control (RBAC)
  • Location hierarchy (tree)
  • Audit rule configuration
  • Material catalog management
