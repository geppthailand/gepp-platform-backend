📋 System Overview

The AI Waste Audit System is part of Bangkok Metropolitan Administration's (BMA) household waste sorting incentive program. Households that correctly sort their waste according to standards receive discounts on waste collection fees.

🎯 Business Requirements

  • Households must submit images of at least 3 waste types: General, Recyclable, and Organic
  • Hazardous waste is optional, but if submitted, it must be verified for compliance
  • AI validates sorting accuracy and provides correction feedback in Thai language

📝 LLM Output JSON Schema

Gemini returns audit results in a structured JSON format, enabling the backend to parse and convert to human-friendly Thai messages automatically.

// LLM Output Structure { "claimed_type": "recyclable", "audit_status": "reject", // "pass" | "reject" "confidence_score": 0.95, "remarks": [ { "code": "wrong_category", "severity": "critical", // "critical" = reject, "minor" = warning "details": { "detected": "organic waste", "reason": "visible food scraps on container" }, "correction_action": "Remove food scraps and wash the container." } ] }

📊 Field Definitions

Field Type Description
claimed_type string Waste type claimed by user (recyclable, general, organic, hazardous)
audit_status enum "pass" = approved, "reject" = needs correction
confidence_score float AI confidence level (0.0 - 1.0)
remarks[].code string Remark code for mapping to Thai template
remarks[].severity enum "critical" = must fix, "minor" = warning only
remarks[].details object Additional data for generating response message

✅ Check Cases by Material Type

♻️ Recyclable Waste

Focus: Cleanliness and material purity

Status Remark Code Condition Details Required
⚠️ Warning recyclable_minor_stain Minor water stains or dust, labels still attached -
✅ Pass recyclable_flattened Cans/boxes are crushed (good for space saving) -
❌ Reject wrong_category Contains general waste (tissues, straws) or organic {"detected": "general waste"}
❌ Reject recyclable_dirty Food residue/grease/liquid inside containers {"substance": "food grease"}
❌ Reject hazardous_contamination Spray cans, batteries mixed in {"item": "spray can"}

🗑️ General Waste

Focus: No valuable or hazardous items mixed in

Status Remark Code Condition Details Required
⚠️ Warning general_contains_recyclable Contains 1-2 recyclable items (bottles/cans) {"item": "plastic bottle"}
❌ Reject hazardous_contamination Light bulbs, batteries, electronics present {"item": "light bulb"}
❌ Reject wrong_category Large amount of food waste (should go to organic) {"detected": "organic waste"}
❌ Reject construction_waste Bricks, cement, large wood (requires special pickup) -

🍂 Organic Waste

Focus: Compostability

Status Remark Code Condition Details Required
⚠️ Warning organic_minor_plastic Small fruit stickers or plastic labels present -
⚠️ Warning organic_hard_bone Large bones that decompose slowly -
❌ Reject organic_in_plastic_bag Food waste tied inside a plastic bag -
❌ Reject organic_contamination Plastic spoons, rubber bands, synthetic materials mixed in {"item": "plastic spoon"}

☠️ Hazardous Waste

Focus: Safety and proper containment

Status Remark Code Condition Details Required
⚠️ Warning hazardous_mixed Batteries mixed with light bulbs (should be grouped) -
❌ Reject hazardous_unsafe Broken glass/tubes without protective wrapping {"reason": "broken glass exposed"}
❌ Reject hazardous_leaking Batteries or chemical bottles are leaking -

🗣️ Thai Response Lookup Table

Mapping table between LLM Remark Codes and natural Thai response messages with template variables.

wrong_category

Used when: Waste doesn't match claimed type

"The attached image is incorrect. Our system detected {detected}. Please sort again correctly."
recyclable_dirty

Used when: Recyclable has contamination

"Recyclable waste did not pass inspection due to {substance} residue. Please wash and dry before resubmitting."
recyclable_minor_stain

Used when: Minor staining (passed with advice)

"Passed! But next time, please rinse off minor stains for optimal recycling quality. 💚"
hazardous_contamination

Used when: Hazardous item in wrong bin

"Danger! Found {item} mixed in. This type of waste must be separated into the hazardous waste bin for collector safety. 🚨"
general_contains_recyclable

Used when: Recyclable items in general waste

"Passed, but too bad! System found {item} that could be recycled. Next time, separate them to earn points! ♻️"
organic_in_plastic_bag

Used when: Food waste in plastic bag

"Organic waste did not pass because it is tied in a plastic bag. Please empty contents directly without the bag. 🍃"
hazardous_unsafe

Used when: Unsafe hazardous packaging

"Did not pass due to {reason} which could injure workers. Please wrap in newspaper or clearly label. ⚠️"
construction_waste

Used when: Construction debris detected

"Sorry, the system does not support construction waste. Please contact your district office to arrange special collection. 🏗️"

🧠 Gemini System Prompt

System prompt configuration for Gemini 2.5 Flash-Lite waste auditing behavior.

🔧 System Prompt Configuration

Role: You are an AI Waste Auditor for the Bangkok Metropolitan Administration (BMA).

Task: Analyze the image of waste submitted by a household. 
Compare the image content against the "Claimed Material Type".

Input:
- Claimed Type: {claimed_type} (e.g., recyclable, general, organic, hazardous)
- Image: [Image Data Base64 or URL]

Rules:
1. Identify all visible items in the image
2. Check for contamination (food, grease, liquid residue)
3. Check for safety hazards (sharp objects, leaking chemicals)
4. Determine if the waste matches the Claimed Type
5. Provide correction actions if rejection

Output Format (JSON Only):
Return a JSON object with:
- status: "pass" or "reject"
- confidence_score: 0.0 - 1.0
- remarks: array of {code, severity, details, correction_action}

Valid Remark Codes:
- wrong_category, recyclable_dirty, recyclable_minor_stain
- hazardous_contamination, organic_in_plastic_bag, organic_contamination
- general_contains_recyclable, hazardous_unsafe, hazardous_leaking
- construction_waste

Example Scenarios:
• Claimed="recyclable" + Greasy pizza box → reject, recyclable_dirty, {"substance": "grease"}
• Claimed="general" + Spray can visible → reject, hazardous_contamination, {"item": "spray can"}
• Claimed="organic" + Food in plastic bag → reject, organic_in_plastic_bag
