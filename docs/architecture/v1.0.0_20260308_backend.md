## 2. Request Lifecycle

```
1. AWS Lambda receives HTTP event
          │
2. app.main(event, context)
          │
3. Parse path: extract deployment_state, route segments
          │
4. CORS check (if OPTIONS method)
          │
5. Route matching:
   ├── Public routes (no auth): /health, /api/auth/login, /api/auth/register,
   │   /api/input-channel/{hash}/*, /documents/*
   │
   ├── Channel-authenticated: /api/input-channel/{hash}/submit (QR token)
   │
   ├── Device-authenticated: /api/auth/iot-devices/login (device token)
   │
   └── Protected routes (JWT required):
       │
       6. Extract Bearer token from Authorization header
       │
       7. verify_jwt_token() → decode payload → get user_id, org_id
       │
       8. Route to appropriate handler:
          ├── auth_handlers.handle_*()
          ├── user_handlers.handle_*()
          ├── organization_handlers.handle_*()
          ├── materials_handlers.handle_*()
          ├── transaction_handlers.handle_*()
          ├── transaction_audit_handlers.handle_*()
          ├── reports_service.*()
          ├── gri_service.*()
          ├── traceability_service.*()
          └── ...
          │
       9. Handler parses request body/params
          │
       10. Handler calls Service method
           │
       11. Service executes business logic
           ├── Query database via SQLAlchemy
           ├── Call external APIs (S3, Vertex AI)
           └── Return result dict
           │
       12. Handler formats response:
           { "statusCode": 200, "body": json.dumps(result) }
```

## 3. Service Layer Pattern

```python
def handle_create_transaction(db_session, body, user_id, organization_id):
```

## 5. Error Handling

```python
# Exception hierarchy
APIException (500)              # Base exception
├── BadRequestException (400)   # Invalid input
├── UnauthorizedException (401) # Auth failure
├── ForbiddenException (403)    # Permission denied
├── NotFoundException (404)     # Resource not found
├── ValidationException (422)   # Validation error
├── ConflictException (409)     # Duplicate/conflict
└── InternalServerException (500) # Server error

# Usage in handlers
try:
    result = service.do_something(...)
    return {"statusCode": 200, "body": json.dumps({"success": True, "data": result})}
except NotFoundException as e:
    return {"statusCode": 404, "body": json.dumps({"success": False, "message": str(e)})}
except APIException as e:
    return {"statusCode": e.status_code, "body": json.dumps({"success": False, "message": str(e)})}
```

## 6. Database Layer

```python
# database.py - Singleton DatabaseManager
class DatabaseManager:
    _instance = None

    def __init__(self):
        self.engine = create_engine(
            DATABASE_URL,
            poolclass=QueuePool,
            pool_size=5,
            max_overflow=10,
            pool_pre_ping=True
        )
        self.SessionLocal = sessionmaker(bind=self.engine)

    @contextmanager
    def get_session(self):
        session = self.SessionLocal()
        try:
            yield session
            session.commit()
        except Exception:
            session.rollback()
            raise
        finally:
            session.close()
```

```python
class PromptLoader:
    def load_ai_audit_prompt(self, rule_set="default", waste_type="general"):
        path = f"ai_audit/{rule_set}/{waste_type}.yaml"
        return load_prompt(path)

    def get_ai_audit_prompts_for_rule_set(self, rule_set="default"):
        return {
            "general": self.load_ai_audit_prompt(rule_set, "general"),

        }
```

## 10. Standard Response Format

```python
# Success response
{
    "statusCode": 200,
    "headers": {"Content-Type": "application/json", "Access-Control-Allow-Origin": "*"},
    "body": json.dumps({
        "success": True,
        "data": {...},
        "message": "Optional message"
    })
}

# Error response
{
    "statusCode": 400,
    "headers": {"Content-Type": "application/json"},
    "body": json.dumps({
        "success": False,
        "message": "Error description",
        "error_code": "VALIDATION_ERROR"
    })
}

# PDF response (raw binary)
{
    "statusCode": 200,
    "headers": {"Content-Type": "application/pdf"},
    "body": base64_encoded_pdf,
    "isBase64Encoded": True
}
```
