```

4. CORS check (if OPTIONS method)

5. Route matching:
   в”њв”Ђв”Ђ Public routes (no auth): /health, /api/auth/login, /api/auth/register,
   в”‚   /api/input-channel/{hash}/*, /documents/*
   в”‚
   в”њв”Ђв”Ђ Channel-authenticated: /api/input-channel/{hash}/submit (QR token)
   в”‚
   в”њв”Ђв”Ђ Device-authenticated: /api/auth/iot-devices/login (device token)
   в”‚
   в””в”Ђв”Ђ Protected routes (JWT required):
       в”‚
       6. Extract Bearer token from Authorization header
       в”‚
       7. verify_jwt_token() в†’ decode payload в†’ get user_id, org_id
       в”‚
       8. Route to appropriate handler:
          в”њв”Ђв”Ђ auth_handlers.handle_*()

          в”њв”Ђв”Ђ organization_handlers.handle_*()

          в””в”Ђв”Ђ ...
          в”‚
       9. Handler parses request body/params
          в”‚
       10. Handler calls Service method
           в”‚
       11. Service executes business logic
           в”њв”Ђв”Ђ Query database via SQLAlchemy
           в”њв”Ђв”Ђ Call external APIs (S3, Vertex AI)
```

## 6. Database Layer

```python
# database.py - Singleton DatabaseManager
class DatabaseManager:
    _instance = None

    def __init__(self):
        self.engine = create_engine(
            DATABASE_URL,
            poolclass=QueuePool,
            pool_size=5,
            max_overflow=10,
            pool_pre_ping=True
        )
        self.SessionLocal = sessionmaker(bind=self.engine)

    @contextmanager
    def get_session(self):
        session = self.SessionLocal()
        try:
            yield session
            session.commit()
        except Exception:
            session.rollback()
            raise
        finally:
            session.close()
```
